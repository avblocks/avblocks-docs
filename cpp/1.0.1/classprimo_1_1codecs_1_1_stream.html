Detailed Description

The Stream interface can be used to implement a data source or sink.

It can be used as an alternative to files when it is not convenient or possible to use files.

virtual bool_t read ( void *  buffer,
uint32_t  bufferSize,
uint32_t *  totalRead 
) [pure virtual]

Reads data from the stream.

Parameters:
buffer[in] A pointer to the buffer where the data shall be stored.
bufferSize[in] Buffer size in bytes. Specifies how many bytes shall be read from the stream.
totalRead[out] A pointer to a variable that stores tha actual number of bytes read from the stream. When the number of bytes returned (totalRead) is less than the requested bytes (bufferSize) then this is considered to be the end of the stream (EOS). Reading at the end of the stream succeeds but 0 bytes are returned.
Returns:
TRUE The operation is sucessful. FALSE Could not read from the stream.
