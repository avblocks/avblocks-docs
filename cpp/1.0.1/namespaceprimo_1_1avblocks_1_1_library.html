Detailed Description

Initializes AVBlocks and manages license information.

This namespace contains global functions for the AVBlocks library. These are C++ wrappers of the functions in the pure "C" interface exported by the library.

Parameters:

Returns:

The returned object has a capacity that has been specified by the capacity parameter. It contains no data, i.e. dataSize() == 0.

bool_t primo::avblocks::Library::isLicenseResolved ( )

Checks whether the current license is resolved.

Returns:
TRUE The internal license status is permanent. It may be changed only by a call to AVBlocksSetLicense. FALSE The internal license status is temporary and may change any moment. Until the current license is resolved AVBlocksSetLicense will reject any license string and will return FALSE.
See also:
setLicense
bool_t primo::avblocks::Library::setLicense ( const char *  license)

Sets AVBlocks license information.

Parameters:
licenseLicense information as string. It is allowed to set NULL in which case the license is cleared.
Returns:
TRUE If the license information (the whole or part of it) is accepted by the library or when the license is cleared. FALSE The license information is not accepted by the library. The new license may not be accepted either because it is invalid or the current license is not resolved yet.
See also:
isLicenseResolved
