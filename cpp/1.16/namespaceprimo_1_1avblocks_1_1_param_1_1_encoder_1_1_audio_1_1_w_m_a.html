const char Quality[] = "ENCODER_WMA_QUALITY"
static

Windows Media Audio: Specifies the quality of the compressed audio.

This parameter is an integer.

Supported values range: from 1 to 100. The default value is 50.

Note
The WMA quality parameter is used only when the encoded bitrate mode is BitrateMode::VBR, otherwise it is ignored.
