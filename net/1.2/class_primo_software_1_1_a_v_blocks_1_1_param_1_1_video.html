string FrameRateFilter = "VIDEO_FRAME_RATE_FILTER" [static]

Video: Specifies whether a video frame rate filter is inserted in the processing chain.

The filter converts the input frame rate to the output frame rate.

This parameter is a boolean value stored as one of the following: System.Boolean (true/false), integral value (0/1) or string ("0","1"). There is no default value.

If true the frame rate filter is always inserted in the processing chain.

If false the filter is never used regardless of the input and output frame rates.

If this parameter is not specified the frame rate filter is used only when the input frame rate differs form the output frame rate.
